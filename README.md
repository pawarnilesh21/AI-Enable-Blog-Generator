AI-Enable-Blog-Generator is a full-stack web application developed using React.js, Node.js, Express.js, and MongoDB Atlas. It enables admins to securely log in via JWT authentication, create blogs with titles, images, and categories, and manage their publish/unpublish status.

The app integrates Gemini API for AI-based blog description generation and ImageKit API for seamless image uploads. Users can browse, search, and comment on published blogs in real time.
